메모리는 평문이라 노출 위험이 있지만 그 위험은 전송 경로마다 다르다. git은 공개 저장소로 푸시될 수 있어 공개 노출이 실재하므로, 비밀정보(자격증명·토큰·키) 저장 금지 경고의 문구는 git 전제를 유지한다. git 외 동기화(Obsidian Sync, Google Drive 등)는 사용자 자신의 계정 안 사적 저장이라고 가정하므로, 이를 이유로 경고에서 git 전제를 제거해 일반화하지 않는다. 이 사적 저장 가정이 깨지는 새 전송 경로를 추가할 때만 경고 범위를 다시 검토한다.
