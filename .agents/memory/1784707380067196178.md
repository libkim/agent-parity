README 소개 영상 작업 체크포인트(2026-07-22): 사용자가 README에 넣을 소개 영상을 Lottie로 만들기로 함. 준비 완료 — ① Lottie Creator MCP를 `claude mcp add lottiefiles-creator -- npx -y @lottiefiles/creator-mcp@latest`로 등록(Claude local 스코프, ~/.claude.json — 프로젝트 .mcp.json 아님, 다른 에이전트/머신엔 미공유) ② `npx skills add LottieFiles/motion-design-skill`로 motion-design 스킬을 .agents/skills/motion-design에 설치(전 에이전트 공유 + Claude symlink). 남은 작업: 새 세션에서 lottiefiles-creator 도구가 로드되면 소개 애니메이션 제작. 이 MCP는 브라우저에서 creator.lottiefiles.com이 열려 있어야 실제 편집 가능(로컬 브리지 방식). 디자인 제약: 다크 배경 로고는 석회석 #d3c8b0 톤 사용(기존 brand 메모리), 리포엔 이미 assets/demo-{dark,light}.{gif,mp4} 데모가 있으므로 소개 영상은 이와 구분되는 별도 자산.
