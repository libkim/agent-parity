agent-parity 릴리스 CI 회귀 테스트를 추가했다. release.yml은 Linux/Windows 사전 게이트에서 Go 기본/configeditor 테스트, release asset 및 체크섬/내장 버전, Unix uninstall/marker/README식 설치, PowerShell 구문·원자적 쓰기·marker·Claude wrapper·Windows install/update/offline uninstall, Windows Git Bash 전체 관리명령 dispatch를 검증한다. 릴리스 후에는 README의 latest 설치 명령을 Linux와 Windows에서 실제 실행하는 smoke job이 있다. 로컬에서 PowerShell 전체 묶음과 Git Bash dispatch가 통과했다. 현재 프로젝트의 기존 .agents 설치 산출물과 루트 .gitignore는 이 작업에서 갱신 대상으로 삼지 않았다.
