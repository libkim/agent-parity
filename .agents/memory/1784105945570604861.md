agent-parity(리포 libkim/agent-parity, 로컬 /home/neo1/my-project/memory-mcp-server)는 Claude Code·Codex·Cursor·Antigravity가 메모리·스킬·지침을 공유하게 하는 '코드형 환경(Environment as Code)' 도구다. 배포 구조: install.sh/ps1 + 템플릿은 RAW(raw.githubusercontent), 바이너리는 GitHub Releases에서 받는다. 버전은 git 태그에서 찍히고, release CI(release.yml)는 오직 v* 태그 push에만 트리거되어 전 플랫폼 정적 바이너리를 빌드한다.
