agent-parity 거버넌스 확정: 설치는 공유 MCP를 단일 메모리 경로로 사용하기 위해 Claude `autoMemoryEnabled=false` 설정 변경을 허용한 것으로 본다. uninstall은 이전 값을 추적·복원하지 않고 `autoMemoryEnabled` 키를 삭제해 Claude 기본 동작으로 되돌린다. 설치 전 사용자가 이미 false로 설정했어도 동일하다. agent-parity memory MCP에 종속된 `enabledMcpjsonServers`의 `memory`와 `mcp__memory__*` 권한도 uninstall 시 제거한다. 이 정책을 `C:\Notes\08 Strategies\크로스 에이전트 메모리\004 크로스 에이전트 메모리 MCP 서버.md`의 거버넌스 섹션에 기록했다.
