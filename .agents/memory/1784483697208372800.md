agent-parity README 편집 규칙 추가: Features의 고정 제품 키워드 `Dependency-free/의존성 없음`, `Non-invasive/비침습`, `Zero-install/무설치`는 구현 설명을 갱신한다는 이유로 임의로 재명명하지 않는다. 2026-07-20 v0.6.1에서 이를 `Runtime-free/별도 런타임 없음`, `Project-scoped configuration/프로젝트 범위 설정`, `Repository-portable/저장소 이식성`으로 바꾼 것은 불필요한 제품 언어 변경이었다. 특히 Zero-install이 현재 config editor provisioning과 충돌하면 문서를 조용히 약화하지 말고, 제품 약속을 유지하도록 코드를 바꿀지 사용자에게 먼저 결정받는다. 문서 감사에서는 사실 오류에 필요한 최소 범위만 수정하고, 제목·마케팅 키워드·어조·대소문자 같은 편집 변경을 섞지 않는다.
