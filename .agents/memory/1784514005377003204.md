v0.6.4 릴리스 완료(2026-07-20) — 거버넌스 전체(수렴형 tombstone 매니페스트, 캐시 GC, prewarm·안내)가 실린 첫 릴리스. 과정: 태그 첫 push는 CI의 zero-install 회귀 테스트에 걸림 — 다른 머신이 만든 테스트 계약("self-heal은 memory-mcp를 다운로드하지 않는다")이 새 거버넌스(pre-warm)와 충돌. 구현이 아니라 테스트를 새 계약으로 갱신(픽스처에 실제 런처 추가, 바이너리 캐시 존재 단언으로 반전, warm 캐시=네트워크 무접촉 불변식은 유지). Windows 테스트는 인프로세스 IWR 목이 별도 프로세스(run.cmd prewarm)에 안 닿아 file:// URI로 전환 — PS 5.1의 IWR file:// 지원 실측 확인. 두 테스트 로컬 통과(Unix sh + native Windows PS 5.1, run.cmd prewarm 실동작 검증 포함) 후 태그 이동 재push, CI 전체 통과, 자산 15개 발행. 실환경 latest-1→latest 검증: 이 리포를 v0.6.3→v0.6.4 update — cache pruned 2, prewarm exit 0. 교훈: 릴리스 게이트 테스트는 거버넌스 변경 시 계약 충돌을 드러낸다 — 테스트의 의도를 확인하고 새 계약을 명시적으로 인코딩할 것.
